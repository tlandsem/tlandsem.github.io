Personal Portfolio Website

// 2025 — 2026 · Web Development

A personal portfolio website built from scratch using Python and Flask. Features a dark cyberpunk aesthetic with an animated network background, falling glyphs, and moving data packets. Includes pages for biography, CV, photography gallery, and Forever Living products. Designed and developed entirely by me — from layout and colour palette to backend routing and Jinja2 templating.

Deployed via Render with a custom domain configured through DNS management — including CNAME record setup and SSL/TLS certificate provisioning for HTTPS enforcement.
