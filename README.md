# trinelandsem.no

Personal portfolio website for Trine Landsem — built with pure HTML, CSS, and JavaScript.

## About
A cyberpunk-themed personal portfolio featuring an animated network background, falling glyphs, and moving data packets.

## Pages
- **Home** — Landing page
- **About me** — Biography and background
- **CV** — Work experience, education, and certifications
- **Projects** — Portfolio of projects
- **Photos** — Photography gallery with lightbox and swipe support
- **Forever Living** — Forever Living products
- **Contact** — Contact form via Formspree

## Tech Stack
- HTML / CSS / JavaScript
- Canvas API (animated background)
- Formspree (contact form)
- GitHub Pages (hosting)
- Cloudflare (DNS & security)
- Custom domain: trinelandsem.no
